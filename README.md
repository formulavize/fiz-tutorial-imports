# fiz-tutorial-imports
Fiz files for teaching imports in the [formulavize](https://github.com/formulavize/formulavize) import tutorial.

These files are served via [GitHub Pages](https://docs.github.com/en/pages).

### Available Imports
- ```@ "https://formulavize.github.io/fiz-tutorial-imports/ant.fiz"```
- ```@ "https://formulavize.github.io/fiz-tutorial-imports/example.fiz"```
- ```@ "https://formulavize.github.io/fiz-tutorial-imports/flying.fiz"```
